# EnSa API
EnSa - Enroute Saviour (Backend)

## Requirements
- Postgres
- Conda
- Python 3.7

## Setup
```
git clone https://github.com/sidshrivastav/shipy_backend.git
conda env create -f environment.yml
cp .env.example .env
```
Get Google Maps API Token for Redirection API

## To run the app
```
python wsgi.py
```
